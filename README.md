A powerful data visualization platform that transforms complex datasets into 
interactive charts and insights. Features custom reporting and export capabilities.

Technical stack:
Frontend:Streamlit (Interactive web interface with real-time updates)
Mapping: Folium + Streamlit-Folium (Interactive maps embedded in web app)
Machine Learning: Scikit-learn (RandomForestRegressor for predictions)
Data Processing: Pandas & NumPy (ETL pipeline and statistical analysis)
Visualization: Plotly Express & Graph Objects (Interactive charts)
Geospatial: Geopy (Geocoding), Polyline (Route encoding/decoding)
APIs: OpenStreetMap (via Nominatim), External traffic data sources
Configuration: Python-dotenv (Environment variable management)
